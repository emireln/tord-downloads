## Download

Get the latest version from [Releases](https://github.com/emireln/tord-downloads/releases/latest).

Available builds:
- **Installer** - `Tord-Setup-x.x.x.exe` (Recommended)
- **Portable** - `Tord-x.x.x-win.zip`
- **x64** - `Tord-x.x.x-x64.exe`
- **ia32** - `Tord-x.x.x-ia32.exe`
